About

Web3 Content Platform adalah aplikasi katalog dan pemutar konten terdesentralisasi yang menghubungkan frontend statis dengan backend pribadi melalui Cloudflare Tunnel, memungkinkan streaming video tanpa server berbayar.

Teknologi

  • Frontend statis dihosting di GitHub Pages
  • Backend Flask berjalan di Termux (Android)
  • Cloudflare Tunnel sebagai reverse proxy HTTPS
  • Vidstack Player untuk pemutaran video
  • Progressive Web App dengan dukungan offline

Versi Aplikasi: 1.0.0

Cara Menjalankan Backend di Termux

  1. Install Python dan cloudflared di Termux
  2. Jalankan ~/web3-backend/start.sh
  3. Salin URL tunnel yang muncul ke kolom di atas
  4. Klik Simpan untuk menghubungkan aplikasi
